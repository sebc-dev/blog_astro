# Débuter avec Astro

Astro est un framework web moderne qui offre des performances ultra-rapides en ne livrant aucun JavaScript par défaut.

## Pourquoi Choisir Astro ?

### Performance d'Abord

- **Zéro JavaScript par défaut** : Ne livrez du JavaScript que quand vous en avez besoin
- **Génération de Site Statique** : Pré-rendu des pages au moment du build
- **Hydratation Partielle** : Hydratez seulement les composants interactifs

### Expérience Développeur

- **Îlots de Composants** : Utilisez votre framework UI préféré
- **Routage Basé sur les Fichiers** : Système de routage simple et intuitif
- **Support TypeScript** : Support TypeScript intégré

## Fonctionnalités Clés

1. **Support Multi-Framework** : React, Vue, Svelte, et plus
2. **Optimisations Intégrées** : Optimisation d'images, bundling CSS
3. **SEO Friendly** : Scores Lighthouse parfaits dès le départ

## Pour Commencer

```bash
npm create astro@latest
cd my-astro-site
npm run dev
```

Commencez à construire des sites web extraordinaires avec Astro dès aujourd'hui !
