# Maîtriser CSS Grid Layout : Guide Complet avec Exemples Pratiques
CSS Grid Layout révolutionne la façon dont nous créons des layouts web. Voici un guide complet pour maîtriser cette technologie puissante et créer des interfaces modernes.
## Introduction à CSS Grid
CSS Grid est un système de layout bidimensionnel qui permet de créer des grilles complexes avec facilité. Contrairement à Flexbox qui est unidimensionnel, Grid excelle dans la création de layouts à deux dimensions.
### Concepts Fondamentaux
```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 20px;
}
```
## Terminologie Essentielle
### Grid Container et Grid Items
```html
<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
</div>
```
### Lignes de Grille (Grid Lines)
- **Verticales** : Colonnes numérotées de 1 à n+1
- **Horizontales** : Rangées numérotées de 1 à n+1
### Pistes de Grille (Grid Tracks)
- **Colonnes** : Espace entre deux lignes verticales
- **Rangées** : Espace entre deux lignes horizontales
## Propriétés du Container
### Définir les Colonnes
```css
.grid {
  /* Colonnes fixes */
  grid-template-columns: 200px 200px 200px;
  /* Colonnes flexibles */
  grid-template-columns: 1fr 2fr 1fr;
  /* Mélange fixe et flexible */
  grid-template-columns: 200px 1fr 100px;
  /* Répétition */
  grid-template-columns: repeat(3, 1fr);
  /* Auto-fit et auto-fill */
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
```
### Définir les Rangées
```css
.grid {
  /* Rangées explicites */
  grid-template-rows: 100px auto 50px;
  /* Rangées implicites */
  grid-auto-rows: minmax(100px, auto);
}
```
### Espacement (Gap)
```css
.grid {
  /* Espacement uniforme */
  gap: 20px;
  /* Espacement différencié */
  row-gap: 10px;
  column-gap: 20px;
  /* Ancienne syntaxe (compatibilité) */
  grid-gap: 20px;
}
```
## Placement des Items
### Placement Basique
```css
.item {
  /* Par numéro de ligne */
  grid-column: 1 / 3; /* De la ligne 1 à 3 */
  grid-row: 2 / 4; /* De la ligne 2 à 4 */
  /* Span (étendre) */
  grid-column: span 2; /* Étendre sur 2 colonnes */
  grid-row: span 3; /* Étendre sur 3 rangées */
  /* Raccourci */
  grid-area: 2 / 1 / 4 / 3; /* row-start / col-start / row-end / col-end */
}
```
### Placement par Nom
```css
.grid {
  grid-template-areas:
    "header header header"
    "sidebar content ads"
    "footer footer footer";
}
.header {
  grid-area: header;
}
.sidebar {
  grid-area: sidebar;
}
.content {
  grid-area: content;
}
.ads {
  grid-area: ads;
}
.footer {
  grid-area: footer;
}
```
## Exemples Pratiques



## Responsive Design avec Grid
### Mobile First
```css
.responsive-grid {
  display: grid;
  gap: 20px;
  /* Mobile : 1 colonne */
  grid-template-columns: 1fr;
}
/* Tablet : 2 colonnes */
@media (min-width: 768px) {
  .responsive-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}
/* Desktop : 3 colonnes */
@media (min-width: 1024px) {
  .responsive-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
/* Large desktop : 4 colonnes */
@media (min-width: 1200px) {
  .responsive-grid {
    grid-template-columns: repeat(4, 1fr);
  }
}
```
### Auto-responsive avec minmax()
```css
.auto-responsive {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 20px;
}
/* S'adapte automatiquement :
   - 1 colonne sur mobile
   - 2-3 colonnes sur tablet
   - 4+ colonnes sur desktop
*/
```
## Techniques Avancées
### Subgrid (CSS Grid Level 2)
```css
.parent-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
.child-grid {
  display: grid;
  grid-column: span 2;
  grid-template-columns: subgrid; /* Hérite de la grille parent */
}
```
### Alignment et Justification
```css
.grid-container {
  /* Aligner tous les items */
  justify-items: center; /* horizontal */
  align-items: center; /* vertical */
  place-items: center; /* raccourci */
  /* Aligner la grille dans le container */
  justify-content: center; /* horizontal */
  align-content: center; /* vertical */
  place-content: center; /* raccourci */
}
.grid-item {
  /* Aligner un item spécifique */
  justify-self: start;
  align-self: end;
  place-self: start end;
}
```
### Grid avec CSS Custom Properties
```css
:root {
  --columns: 3;
  --gap: 20px;
  --min-item-width: 250px;
}
.dynamic-grid {
  display: grid;
  grid-template-columns: repeat(var(--columns), 1fr);
  gap: var(--gap);
}
/* Responsive avec CSS variables */
@media (max-width: 768px) {
  :root {
    --columns: 1;
    --gap: 15px;
  }
}
```
## Compatibilité et Fallbacks
### Support Navigateur
```css
.fallback-layout {
  /* Fallback pour anciens navigateurs */
  display: flex;
  flex-wrap: wrap;
}
/* Amélioration progressive */
@supports (display: grid) {
  .fallback-layout {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  }
}
```

## Conclusion
CSS Grid Layout offre une puissance et une flexibilité inégalées pour créer des layouts modernes :
### Avantages Clés
- **Bidimensionnel** : Contrôle total sur lignes et colonnes
- **Responsive natif** : `auto-fit`, `auto-fill`, `minmax()`
- **Semantic** : Zones nommées pour plus de clarté
- **Puissant** : Layouts complexes avec peu de code
### Quand Utiliser Grid vs Flexbox
- **Grid** : Layouts bidimensionnels, placement précis
- **Flexbox** : Alignement unidimensionnel, distribution d'espace
Maîtriser CSS Grid vous permettra de créer des interfaces web modernes, responsives et maintenables avec une efficacité remarquable !
