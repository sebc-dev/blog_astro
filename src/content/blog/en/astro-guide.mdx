# Getting Started with Astro

Astro is a modern web framework that delivers lightning-fast performance by shipping zero JavaScript by default.

## Why Choose Astro?

### Performance First
- **Zero JavaScript by default**: Only ship JavaScript when you need it
- **Static Site Generation**: Pre-render pages at build time
- **Partial Hydration**: Hydrate only the interactive components

### Developer Experience
- **Component Islands**: Use your favorite UI framework
- **File-based Routing**: Simple and intuitive routing system
- **TypeScript Support**: Built-in TypeScript support

## Key Features

1. **Multi-Framework Support**: React, Vue, Svelte, and more
2. **Built-in Optimizations**: Image optimization, CSS bundling
3. **SEO Friendly**: Perfect Lighthouse scores out of the box

## Getting Started

```bash
npm create astro@latest
cd my-astro-site
npm run dev
```

Start building amazing websites with Astro today!
